Compiling the application:
javac -cp "/usr/local/Cellar/tomcat/8.5.5/libexec/lib/servlet-api.jar:/usr/local/Cellar/tomcat/8.5.5/libexec/lib/mysql-connector-java-5.1.40-bin.jar:/usr/local/Cellar/tomcat/8.5.5/libexec/lib/mongo-java-driver-3.2.2.jar" *.java

Running the application:
1) The application can be run through having the source code folder in the Apache-tomcat-8.5.5/webapps/folder_website_name.
2) After coping the source code folder at the webapps directory, start the apache tomcat, mysql and mongodb server.
3) Run the application link on the browser by the link as "http://localhost:8080/BestDeal/login.html".
